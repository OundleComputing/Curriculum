First Some Terminology

Repository

Repository - Repositories are where you store each of your different projects. For example you might have one Repository called Website which contains a Website you are working on. You might also have another called Python Programs - containing (unsurprisingly!) All of your Python Programs.

Repositories can be created locally (on your computer) and then synchronised with GitHub so that you can work on them elsewhere by cloning and others can do this too

Commands -Creating a Repository (Local)

git init - this creates a local repository. Make sure you are in a suitable directory - probably best to have your own GitHub folder in your H: drive

These 3 commands concurrently will upload all local changes to the selected repository

git add * - this creates a local repository. Make sure you are in a suitable directory - probably best to have your own GitHub folder in your H: drive

git commit -m "comment" - this creates a local repository. Make sure you are in a suitable directory - probably best to have your own GitHub folder in your H: drive

git git push - this creates a local repository. Make sure you are in a suitable directory - probably best to have your own GitHub folder in your H: drive
